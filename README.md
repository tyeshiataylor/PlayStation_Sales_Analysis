## PlayStation_Sales_Analysis
This project analyzes sales data of Playstation to uncover trends, patterns, and insights using data visualization techniques.  The goal was to practice data cleaning, analysis, and creating meaningful visual representations of the Playstation data.

## Dataset
Download Dataset From Kaggle:
https://www.kaggle.com/datasets/gvidalguiresse/playstation-sales-and-metadata-ps3ps4ps5

## Technologies Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook


## How to Run
1. Download or clone the repository.
2. Open the .ipynb file using Jupyter Notebook or VS Code.
3. Run the cells to see the analysis and visualizations.


## Objectives
- Analyze overall sales performance
- Identify top-performing categories
- Examine sales trends over time

4.Total Sales by Release Year

Insights - 
Physical purchases peaked between 2007 and 2010 when discs were the primary option, the market saw a steady decline as high-speed internet made digital downloads more convenient. 
By 2019, sales hit a "digital tipping point," where the rise of the PlayStation Store, subscription services, and digital-only console editions effectively replaced the traditional physical market.
